Project Overview

Comprehensive tool detection system for space station safety

Problem Statement

Space stations require continuous monitoring of essential safety tools including Fire Extinguishers, Toolboxes, and Oxygen Tanks. Manual tracking is prone to human error and may compromise mission safety. Traditional monitoring systems lack real-time capabilities and intelligent alerting mechanisms.

Real-time Monitoring

Need instant detection and verification of tool presence

Safety Critical

Missing tools can compromise crew safety and mission success

Data Tracking

Need comprehensive logging for compliance and analysis

Solution Architecture

Frontend Components
  • HTML5, CSS3, JavaScript (ES6+)
  • Bootstrap 5 for responsive design
  • WebRTC for camera integration
  • Chart.js for data visualization
Backend Components
  • Python Flask web framework
  • YOLOv8 (Ultralytics) for detection
  • MongoDB for data storage
  • OpenCV for image processing

Dataset & Training

Falcon Synthetic Data

Our training dataset was generated using Falcon Editor by Duality AI, a high-quality synthetic data generation platform. This approach provides several advantages:

Unlimited training data generation
Perfect ground truth annotations
Controlled lighting and environment
Cost-effective compared to real data collection
Training Statistics
Training Images: 2,500+
Validation Images: 500+
Epochs: 50
Batch Size: 16
